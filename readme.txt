== Description ==
Product Viewer is an easy to use WooCommerce 3D product variation plugin. It offers an aesthetic and professional experience to select colors for your 3d products. It can turn your product thumbnail to a 3D Model viewer that customers can customize and view.

Product Viewer for WooCommerce is an solution to increase your sells, attract more customers and make your shop modern and elegant, with Product Viewer you turn you WooCommerce shop into interactive experience.



* Easy to Install 
* Easy to Edit/manage










== Frequently Asked Questions ==


= What ProductViewer does?  =

ProductViewer give you an option to display 3D Models as product.

= Can i edit my model inside ProductViewer?  =

no, you need to use complete model,with lights and env objects as you wish.


= Which files ProductViewer is support ?  =

right now we support gltf/glb/fbx files,
in the near future we will add more options

= ProductViewer is support gltf ?  =

yes, we support gltf&fbx
